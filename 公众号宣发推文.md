### 1. 🎯 你的 AI 求职工作流，有几个致命bug？

市面上的 AI 简历工具一抓一大把，但真正能打的有几个？学姐总结了 6 个最要命的"海投翻车现场"：

| 痛点 | 现象 | Vibe CV Copilot 的破局 |
|------|------|------------------------|
| **排版易碎** | AI 生成网页版简历想微调，加粗一个词格式就崩了；LaTeX 门槛又太高 | **无损 Docx 模板注入**：用你手动调好的 Word 底稿，AI 只做"填词游戏"，所见即所得 |
| **AI 幻觉填词** | AI 堆砌"数据基座"等空洞废话，一问细节就露馅 | **HITL 人工审核闭环**：AI 先生成双栏对比 Markdown 草稿，等你手动删改确认后再注入 Word |
| **面试见光死** | 简历写得漂亮，面试官追问两句就卡壳 | **全知视角 Mock QnA 库**：AI 会查阅你的项目代码，还原真实数据，自动生成面试官连环追问武器库 |
| **海投混乱** | 投了 20 家公司，面试前连 JD 都找不到 | **自动 Tracker 绑定**：每次生成简历后，AI 自动将公司、岗位、JD 链接、投递状态写入 Excel 表格 |
| **0 经验困境** | 连个像样的项目都没有，简历无从下笔 | **0-to-1 项目孵化**：输入 JD，AI 帮你从 GitHub 筛选可魔改的开源项目，1天"吃透"一个高含金量项目 |
| **实习做完不会写** | 一堆代码和周报，不知道怎么包装成简历成果 | **1-to-10 成果榨取**：AI 扫描代码仓库，自动提炼 STAR 法则要点，生成面试可能追问的"考点武器库" |

### 2. 🤖 Vibe CV Copilot：9步打造 Headhunter 级简历

这个工具叫 **[Vibe CV Copilot](https://github.com/Esther080310/One-Stop-Resume-Copilot)**，是我用 Trae (Vibe Coding) 从 0 到 1 搭建的全套简历智能体。

整个工作流只有 **9 个核心步骤**，却覆盖了从"素材准备"到"Offer 拿到"的完整闭环：

> **💡 适用场景说明**：只要你的 IDE 支持配置自定义 Skill（比如 **Trae**、**Cursor** 等），就可以把这个工作流直接挂载进去。它不需要你有独立的项目——你完全可以把它配置在**任意一个正在做项目的 IDE** 里，在写代码的间隙，直接对那个项目的简历细节进行精修，顺便把简历投了！

```
输入 JD → 拆解关键词 → XYZ 重构 → HITL 审批 → 无损注入 Word
     → 生成 Cover Letter → 自动登记 Tracker → 生成 Mock QnA
     → 项目孵化（可选）→ 成果复盘（可选）
```

每一份简历都是**你亲手审核过的**，绝不会是那种"AI 一键生成、面试一戳就碎"的废稿。

### 3. 🔥 核心功能逐个拆解
#### ✅ 0-to-1 项目孵化（连项目都没有？AI 帮你造）
输入目标 JD，AI 从 GitHub 筛选 2-3 个技术栈高度匹配的项目，并规划"防撞车改造点"（如加缓存、换微服务）。1天搞定一个有说服力的简历项目。

#### ✅ 1-to-10 实习复盘与成果榨取
实习结束只有一堆琐碎的代码和周报？直接扔给 AI，它会自动扫描数据流和工程难点，降维提炼成 HR 视角的 STAR 法则要点，把平淡的打杂包装成高含金量业务成果。

#### ✅ 无损 Docx 注入（万能排版兼容）
无论你喜欢什么样的排版风格，只要准备一份你自己手动精调好的 Word 简历作为"底稿模板"丢进 `resources/` 文件夹即可！AI 在底层用 `python-docx` 直接修改 XML 文本节点，不破坏任何原有格式，实现真正的万能兼容和所见即所得。

#### ✅ Human-in-the-loop（防 AI 幻觉）
AI 不会直接生成最终简历，而是先在 IDE 里输出一个**双栏对比的 Markdown 草稿** (`review_draft.md`)。你满意了，手动确认，AI 才正式注入 Word。

#### ✅ 全知视角与求职认知库 (Career Knowledge Base)
你可以把平时看小红书博主分享、听宣讲会得到的**零散笔记**丢进它的 `Career_Knowledge_Base/` 文件夹。它会自动吸收为行业认知，并在生成简历时为你提供精准的"行业黑话"。面试前，它还会查阅你的历史项目代码，自动生成针对性的**面试 Mock 问答库**。

#### ✅ 一键生成求职信 (Cover Letter)
简历落定后，不想再费脑细胞写邮件正文？AI 会根据投递场景，自动生成去 AI 味、纯文本排版、且高度提炼 JD 匹配点的**高定版求职信**，你只需 Ctrl+C/V。

#### ✅ 自动 Job Tracker
每次生成简历后，AI 会自动把目标公司、岗位、JD 链接、投递状态写入 `Job_Application_Tracker.xlsx`。海投再也不会失联。
